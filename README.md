# Homebrew Tap for BiliObjCLint

This is the official Homebrew tap for [BiliObjCLint](https://github.com/pjocer/BiliObjcLint).

## Installation

```bash
brew tap pjocer/biliobjclint
brew install biliobjclint
```

## Update

```bash
brew update && brew upgrade biliobjclint
```

## Source Repository

For documentation and source code, visit: https://github.com/pjocer/BiliObjcLint
